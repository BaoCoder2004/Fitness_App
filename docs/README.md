# Tài liệu hệ thống (Guides)

## Danh sách tài liệu
- `kien_truc.md`: Kiến trúc, pattern, cách tổ chức code.
- `co_so_du_lieu.md`: Cấu trúc Firestore, bảng/chỉ mục và mapping vào code.
- `flows.md`: Các luồng nghiệp vụ chính (User + Admin) và file code liên quan.

## Cách đọc nhanh
1) Bắt đầu ở `kien_truc.md` để nắm layout dự án và dependency giữa các tầng.  
2) Xem `co_so_du_lieu.md` để hiểu collections/subcollections và entity-model mapping.  
3) Đọc `flows.md` khi cần giải thích luồng code cụ thể trong bảo vệ/đánh giá.
